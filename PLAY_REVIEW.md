## Executive Summary (종합 요약)

Pixel Infinity(Tavern Survivors)는 Construct 3 기반의 뱀파이어 서바이버즈 스타일 로그라이트이다. **런 내 재미의 기본 골격은 잘 갖춰져 있으나**, 밸런스 파괴 요소, 메타 프로그레션 부재, UX 미완성이 게임의 장기 지속력을 심각하게 저해한다.

### 종합 등급

| 영역 | 등급 | 요약 |
|------|------|------|
| 전투 밸런스 | C+ | 무기 간 DPS 편차 17배, 무한 스케일링 악용 가능 |
| 적/난이도 | C | 8종 중 6종 동일 AI, 2종 등장 불가, 데스월 발생 |
| 경제/진행 | D+ | 히어로 전체 해금에 236런(39시간), 메타 진행 부재 |
| UX/게임필 | C | 히트스톱 없음, 디버그키 노출, 설정 미구현 |

## CRITICAL Issues (치명적 - 즉시 수정 필요)

### 1. 데미지 무한 스케일링 (밸런스 파괴)
- **Reaper's Contract**: 킬당 +0.1% 데미지 누적, **상한 없음** → 1000킬 = +100%
- **Hoarder's Charm**: 상자당 +2.5% 데미지 누적, **상한 없음**
- **회피 100% 도달 가능**: Zephyr Amulet(무한 스택) + Evasion Tome = 완전 무적
- **크리 확률/배율 캡 없음**: Cowboy + Focus Crystal + Fractured Lens = 86% 크리, 6x 배율
- **Ultra Crit 20x**: 퍼센트 보너스 이후 적용 → 기본 데미지의 **140배** 가능

**수정안:**
```
percentBonus 캡: 500%
크리 확률 캡: 80%  |  크리 배율 캡: 5x
Ultra Crit: 20x → 10x
Reaper's Contract 캡: +100%  |  Hoarder's Charm 캡: +50%
회피 캡: 75%
```

### 2. 적 2종이 게임에서 절대 등장하지 않음
- **Enemy3 (박쥐)**: 레벨 14 요구 → 보스가 레벨 12에서 게임을 끝냄 → **등장 불가**
- **Enemy2 (고블린)**: 레벨 12 해금이지만 보스 스폰 시 모든 적 제거 → **사실상 못 봄**
- 코드에만 존재하는 2종의 적 = **개발 리소스 낭비**

**수정안:** Enemy3 → 레벨 6, Enemy2 → 레벨 7로 하향

### 3. 데미지 이중 스케일링으로 "데스 월" 발생
- 레벨 스케일링(+25%/레벨) × 시간 스케일링(+0.25x/20초) = **이차함수적 폭증**
- 레벨 12, 5분 시점 Enemy6 데미지: `25 × 3.25 × 4.75 = 386 DPS` → **0.56초 즉사**

**수정안:** 시간/레벨 배율 중 하나만 데미지에 적용, 또는 `max(levelBonus, difficultyMultiplier)` 사용

### 4. 히어로 전체 해금에 236런(39시간) 소요
- LobbyManager 실제 가격: 500~3,000G, **총 13,000G**
- 런당 평균 55G 수입 → 236런 필요
- Vampire Survivors는 20~30런으로 대부분 해금 가능
- **HeroData.js와 LobbyManager.js의 가격이 불일치** (데이터 이중화 버그)

**수정안:** 히어로 최대가격 1,000G로 축소 (총 ~4,000G), 골드 드롭률 20%로 상향

### 5. 디버그 키가 프로덕션에 노출
- **U키**: 즉시 승리 +200골드
- **L키**: 즉시 레벨업
- 플레이어가 발견하면 게임 경제 완전 파괴

**수정안:** 프로덕션 빌드에서 디버그 키 제거 또는 `DEBUG_MODE` 플래그로 보호

### 7. 영웅 밸런스 - 최비싸=최약, 무료=최강
- **Paladin(1000G)**: guardian_shield 데미지 0, 기본 공격 DPS 22.4 → **게임 최약**
- **Cowboy(무료)**: 크리 20% + 3x배율 → 기대 DPS **상시 1.4배**, 아이템으로 **2배+**
- **Vampire(750G)**: 흡혈 1HP/hit → 적 데미지 5~35 대비 무의미
- **Skeleton(600G)**: XP +15% → XP Tome 1스택(+25%)보다 약함

### 8. 보스 전투 단조로움
- 패턴 1개: "플레이어에게 걸어감 + 0.8초마다 직선 화염구"
- 화염구가 직선 이동이라 이동만 하면 100% 회피 → **긴장감 없음**
- 페이즈 전환, 광역기, 소환물 등 **없음**

### 9. 적 AI 단조로움 - 8종 중 6종 동일 패턴
- Enemy1/2/3/5/6/8 전부 "플레이어 방향으로 직진" 동일 AI
- 속도/HP만 다를 뿐 행동 차이 없음 → 중반 이후 단조로움

### 10. 메타 프로그레션 부재
- 영구 스탯 업그레이드 **없음** (VS의 PowerUp에 해당)
- 무기 진화/합성 **없음** (VS의 핵심 매력)
- 업적 시스템 **없음**
- 컬렉션/도감 **없음**
- 난이도 모드 **없음**
- **"다시 하고 싶은 이유"가 히어로 해금뿐**

### 11. UX - 히트스톱 없음
- 적 처치/피격 시 **프레임 프리즈(히트스톱)가 전혀 없음**
- VS류 게임의 핵심 쾌감 요소 부재
- 카메라 쉐이크도 0.15초로 거의 느끼지 못함

### 12. 레벨업 시 UPGRADES 배열 미사용
- `GameConfig.js`에 기본 스탯 업그레이드 4종이 정의되어 있으나 **연결 안 됨**
- 레벨업이 무기/토메 선택만 제공 → 기본 스탯 성장 경로 없음

## Positive Findings (긍정적 요소)

1. **모듈러 아키텍처** - 30개 ES6 모듈로 깔끔한 코드 분리
2. **가산적 데미지 시스템** - 곱셈 폭발 방지 설계 (캡만 추가하면 됨)
3. **다양한 무기 메커니즘** - 궤도/유도/도탄/폭발/경로 등 20+종
4. **아이템 시너지 설계** - 25종 아이템의 스태킹 조합이 빌드 다양성 제공
5. **반응형 스케일링** - 다양한 해상도 대응
6. **실버 경제** - 상자 비용 증가 곡선이 적절 (5~6개/런)
7. **초반 실버 보장** - 첫 4킬에서 2실버 드롭 보장
8. **레벨업 연출** - easeOutBack 스케일 애니메이션이 폴리시 있음
9. **보스전 1v1 모드** - 적 클리어 후 보스만 등장하는 연출이 좋음
10. **세이브 안전성** - 골드 획득 시 즉시 저장
